# Web Design Fundamentals Short Certificate

**Type:** Short Certificate
**Length:** 4 courses
**Path:** Web track starting point

---

## What This Is

The Web Design Fundamentals Short Certificate gets you from zero to building
real web pages in four courses. You will learn how websites are structured,
how they are styled, and how to make them interactive with code.

It is also the foundation for the Web Development Certificate and the Digital
Design Certificate. Everything here counts toward those longer credentials.

---

## The Four Courses

| [light] | |
|---|---|
| CS 110 – Intro to Computers & Applications | Computer basics, file management, working in a professional software environment |
| BUS 261 – User Interface Design | Planning websites and apps for users — how to make things easy to use, not just easy to build |
| CS 251 – HTML & CSS | The building blocks of every website: HTML for structure, CSS for style and layout |
| CS 140 – JavaScript I | Making web pages interactive — buttons that do things, forms that respond, content that changes |

---

## What You Will Be Able to Do

By the time you finish this short certificate, you will be able to:

- Plan a website with users in mind — thinking through layout, navigation,
  and usability before writing a single line of code
- Build a complete web page using HTML and style it with CSS
- Add basic interactivity using JavaScript
- Understand how the front-end of a website works and how the pieces fit together

---

## Who This Is For

This certificate is a strong starting point if you:

- Are curious about how websites work under the hood
- Want to write code — even if the idea is new or a little intimidating
- Like solving problems that have clear, testable answers
- Are not yet sure how far you want to go and want to earn something real
  while you figure that out

You do not need any previous coding experience. CS 110 and BUS 261 are
designed for students who are starting from the beginning.

---

> alert: tip
>
> **A lot of students are nervous about writing code.** That feeling is normal and it goes away faster than you expect. You start with very small pieces — write a few lines, see what happens, adjust. The instructions are step-by-step. Your instructor is nearby. You are not expected to figure it out on your own. Most students who thought coding would be too hard end up surprised by how quickly it starts to make sense.

---

## Jobs This Prepares You For

Short-certificate graduates with web skills can pursue or build toward:

- Entry-level web content editor
- Website assistant or maintainer
- Freelance front-end work (basic sites, landing pages)
- Junior developer apprenticeship or internship

Continuing into the one-year certificate adds significantly more capability
and job options.
