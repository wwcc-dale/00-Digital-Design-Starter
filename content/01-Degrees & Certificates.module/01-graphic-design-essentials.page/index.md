# Graphic Design Essentials Short Certificate

**Type:** Short Certificate
**Length:** 4 courses
**Path:** Design track starting point

---

## What This Is

The Graphic Design Essentials Short Certificate is the fastest way to earn a
credential in this program. Four courses. Real tools. Skills you can put to
use right away.

It is also the foundation for the Graphic Design Certificate and the AAS
degree. If you go further, everything you do here counts.

---

## The Four Courses

| [light] | |
|---|---|
| CS 110 – Intro to Computers & Applications | Computer basics, file management, working in a professional software environment |
| BUS 263 – Photoshop | Photo editing, image compositing, digital retouching, working with layers and masks |
| CS 224 – Computer Illustration | Vector graphics, logo creation, illustration, working with paths and shapes in Illustrator |
| CS 220 – Two-Dimensional Design | Core design principles: color theory, composition, balance, hierarchy, and visual communication |

---

## What You Will Be Able to Do

By the time you finish this short certificate, you will be able to:

- Open, edit, and export professional-quality images in Photoshop
- Create logos and scalable vector graphics in Illustrator
- Apply design principles to layouts — not just make things that look nice,
  but make things that communicate clearly
- Work with industry-standard software confidently and professionally

---

## Who This Is For

This certificate is a strong starting point if you:

- Are drawn to visual work — creating things that look good and communicate well
- Want to learn Photoshop and Illustrator, which appear on job listings
  throughout the design and marketing industries
- Are not yet sure how far you want to go in the program and want to earn
  something real while you figure that out

---

## Jobs This Prepares You For

A short certificate alone may not qualify you for every design job, but it
demonstrates real skills. Roles that short-certificate graduates can pursue
or build toward:

- Entry-level production designer or layout artist
- Print shop operator
- Social media content creator
- Freelance designer (logo work, flyers, basic branding)
- Design assistant

A stronger portfolio — built by continuing into the one-year certificate —
opens more doors.

---

## Where This Leads

```
Graphic Design Essentials  ──►  Graphic Design Certificate (1-year)
                            ──►  Digital Design Certificate (1-year)
                            ──►  AAS Degree (2-year)
```

All four courses in this short certificate count toward any of those
longer credentials. You are not starting over — you are building.
