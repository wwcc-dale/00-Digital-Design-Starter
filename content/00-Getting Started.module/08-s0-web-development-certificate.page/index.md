# Web Development Certificate

**Type:** One-Year Certificate
**Length:** ~60 credits
**Focus:** Code-heavy — front-end, back-end, databases, full-stack applications
**Capstone:** CS 231 – Capstone Application Development (a working web app)

---

## What This Is

The Web Development Certificate takes you from the basics of building web
pages all the way through building full-stack applications — programs that
have a user interface on the front end, logic running on a server, and a
database storing and managing information.

This certificate is for students who want to build things with code. It is
the path to front-end and full-stack developer roles.

## The Capstone: CS 231

This capstone is a full working application — not just a website. You will
build something that stores and retrieves data, manages user input, and
handles real logic.

**Examples of capstone applications:**

- A task management app that stores tasks in a database, lets users add,
  edit, and delete them, and organizes them by category or deadline
- An inventory system for a small business to track products and quantities
- A recipe database where users can search, save, and add recipes
- A community resource finder that displays services by location or type

The capstone proves you can build complex, multi-layer software similar to
what professional developers create at work.

---

## The Progression of the Code Courses

One of the most satisfying things about this certificate is watching your
own capability grow:

- **CS 251 / CS 140** — You build static pages that display content
- **CS 262** — You learn how to work like a professional developer
- **CS 235** — You learn how applications store and retrieve information
- **CS 240** — You learn to build applications that talk to servers and APIs
- **CS 247** — You build modern interfaces using professional frameworks
- **CS 231** — You put all of it together into one working application

Each course builds directly on the last. By the end, you are not just
writing code — you are building software.

---

## Who This Is For

This certificate is a strong fit if you:

- Want to write code and build applications
- Are drawn to problem-solving and logic
- Are aiming for front-end or full-stack developer roles
- Want to work in tech — at a company, remotely, or freelancing

You do not need programming experience. The certificate starts from the
beginning and builds systematically.
