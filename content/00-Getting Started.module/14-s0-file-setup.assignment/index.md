# File Directory Setup

- stats
- 25 | Points | success

## Why This Matters

Imagine finishing a project and not being able to find it when you need to
submit it. Or coming back to a project after two weeks away and not knowing
what any of your files are called. Good file organization prevents both.

A consistent folder structure also signals professionalism. During your
capstone and in job interviews, you may be asked to share your work files.
A clean, organized folder structure makes a strong first impression.

---

## Skills You Need

This assignment requires only basic computer skills: navigating folders,
creating new folders, and naming them. No design software or code is needed.

---

## The Folder Structure You Will Build

Create the following folder structure exactly as shown. Use the exact
capitalization and spelling below.

```
[YourLastName]_[FirstInitial]/
├── 01_Graphic-Design/
│   ├── Photoshop/
│   ├── Illustrator/
│   └── Two-D-Design/
├── 02_Web-Development/
│   ├── HTML-CSS/
│   ├── JavaScript/
│   └── Projects/
├── 03_Portfolio/
│   ├── Work-in-Progress/
│   └── Final-Work/
└── 04_Resources/
```

**Your root folder name:** Replace `[YourLastName]` with your actual last
name and `[FirstInitial]` with the first letter of your first name.
No spaces — use the underscore.

Example: If your name is Maria Garcia, your root folder is `Garcia_M`.

Everything you create in this program goes inside this root folder.

---

## Step-by-Step Instructions

- steps: Step-by-Step Instructions
- Navigate to the shared drive class folder in File Explorer. Ask your instructor for the exact location if you are not sure.
- Create your root folder: name it `LastName_Initial` (example: `Garcia_M`). Use only letters and underscores — no spaces.
- Inside your root folder, create four top-level folders: `01_Graphic-Design`, `02_Web-Development`, `03_Portfolio`, `04_Resources`.
- Add sub-folders: inside `01_Graphic-Design` create `Photoshop`, `Illustrator`, `Two-D-Design`. Inside `02_Web-Development` create `HTML-CSS`, `JavaScript`, `Projects`. Inside `03_Portfolio` create `Work-in-Progress` and `Final-Work`.
- Take a screenshot using Snipping Tool (Windows key → type "Snipping Tool" → New → drag to select). Save it as `LastName_Initial_folder-setup.png` inside your `04_Resources` folder. Your screenshot must show the root folder name, all four top-level folders, and at least one expanded to show sub-folders.
- Upload your screenshot to this assignment in Canvas.

> alert: warning
>
> Do not create your folder on the Desktop. Desktop files are not reliably saved and may be lost if the computer is reset. Always use the shared drive.

## Success Criteria

- checklist: Success Criteria
- Folder is on the shared drive, not the Desktop
- Root folder uses correct format (LastName_Initial)
- All four top-level folders present and correctly named
- All sub-folders present inside each top-level folder
- Names use hyphens instead of spaces where required
- Screenshot clearly shows the structure
- Screenshot file named correctly and submitted

## Stretch Levels

- accordion: Stretch Levels
- Base — Required for everyone
  - Complete the folder structure as described and submit the screenshot. All students should reach this level.
- Intermediate — If you finish Base with time remaining
  - Add one more level inside `03_Portfolio/Work-in-Progress`. Create a sub-folder for each course you plan to take — for example: `Photoshop`, `Illustrator`, `HTML-CSS`. Include the updated screenshot in your submission.
- Advanced — Challenge yourself
  - Research professional naming conventions used in graphic design studios or web development teams. Find at least one example of how real studios or freelancers organize their project files. Add a plain-text file called `naming-notes.txt` inside `04_Resources` with a 3–5 sentence description of what you found and how it compares to the structure you built here. Include this file alongside your screenshot.
