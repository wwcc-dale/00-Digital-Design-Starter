# Digital Design Certificate

**Type:** One-Year Certificate
**Length:** ~45 credits
**Focus:** Balanced — design skills and web skills together
**Capstone:** BUS 288 – Web Building Capstone (a launched website)

---

## What This Is

The Digital Design Certificate is the balanced path. You will develop both
graphic design skills and web development skills, then bring them together
in a final capstone project — a complete, working website you build from
start to finish.

This certificate is for students who want a broad skill set, or who are not
yet sure whether they prefer design or code and want to develop both before
deciding.

## The Capstone: BUS 288

This is the course where everything comes together. You will:

1. Choose a real-world website purpose (a small business, portfolio, community
   resource, or similar)
2. Research your users and plan the site structure
3. Create wireframes and visual designs
4. Build the site using HTML, CSS, JavaScript, and responsive design
5. Test it with real users and improve it based on their feedback
6. Launch it to a live server

The result is a published website in your portfolio — proof that you can
carry a web project from idea to launch.

## Who This Is For

This certificate is a strong fit if you:

- Want to be able to handle both design and development tasks
- Are not yet sure which direction you prefer and want to try both
- Are aiming for roles that require visual skills and technical skills together
- Have a shorter timeline and want to earn a one-year credential that covers
  a lot of ground
